# ⛽ Petrol Station Management System

A full-stack web-based system designed to manage petrol station operations in real time, ensuring transparency, accountability, and efficient business management.

## 🚀 Features

- Real-time fuel sales tracking
- Automatic fuel consumption calculation
- Daily, weekly, and monthly reports
- Debt management (paid & unpaid tracking)
- Role-based access control (CEO, Manager, Attendant)
- Secure authentication with JWT
- Automatic receipt generation
- Short/loss detection and recording
- Multi-branch system with centralized monitoring
- Network integration using Tailscale

## 🛠️ Tech Stack

### Frontend:
- React.js
- TypeScript
- CSS

### Backend:
- Java (Spring Boot)
- REST APIs
- JWT Authentication

### Database:
- MySQL

### Tools:
- Git & GitHub
- Postman

## 🌐 System Architecture

- Full-stack application (Frontend + Backend)
- Role-based dashboards
- Secure API communication
- Distributed branch system with centralized CEO access

## 💡 Key Impact

- Improved business transparency
- Reduced manual errors
- Enabled remote monitoring of multiple branches
- Reduced operational costs (less physical supervision)
